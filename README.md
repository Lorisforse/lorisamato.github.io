# loris-cv

Personal CV site built with [Hugo](https://gohugo.io/) and the [Gokarna](https://github.com/526avijitgupta/gokarna) theme.

Live at **https://lorisamato.github.io/**

## Local development

```bash
hugo server -D
```

Open http://localhost:1313/

## Deploy

Push to `main` — GitHub Actions builds and deploys automatically to GitHub Pages.

Workflow: `.github/workflows/hugo.yml`
